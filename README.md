<h1>Smart Classroom: Live Lecture to Auto Notes</h1>

<h2>Overview</h2>
<p>
Smart Classroom is an intelligent lecture transcription system designed to automatically convert live classroom lectures into structured digital notes. 
The system captures lecture audio, processes it using advanced speech recognition models, and stores the generated text in a database so that students can review lecture content later.
</p>

<p>
The project aims to improve learning accessibility by ensuring that students do not miss important lecture content and can revisit the generated notes anytime.
</p>

<h2>Key Features</h2>
<ul>
<li>Real-time speech-to-text conversion of live classroom lectures</li>
<li>Automatic lecture note generation from spoken audio</li>
<li>Backend APIs for processing and managing lecture transcripts</li>
<li>Database storage for saving generated lecture notes</li>
<li>Retrieval and management of lecture notes for later review</li>
<li>Scalable architecture designed for classroom-level deployment</li>
</ul>

<h2>Technology Enhancements</h2>
<ul>
<li>Integration of transformer-based speech recognition models</li>
<li>Implementation of automated lecture transcription pipeline</li>
<li>Backend REST API architecture for scalable processing</li>
<li>Database-backed storage system for structured note management</li>
<li>System designed for IoT-based lecture capture in smart classrooms</li>
</ul>

<h2>Technologies Used</h2>
<ul>
<li>Python</li>
<li>FastAPI (Backend API Development)</li>
<li>MySQL (Database)</li>
<li>SQLAlchemy (ORM for database interaction)</li>

<li>Hugging Face Transformers</li>
<li>React.js (Frontend development - in progress)</li>

</ul>
