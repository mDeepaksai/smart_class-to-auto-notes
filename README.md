 Developing a system that converts live classroom lectures into automatically generated digital notes using speech processing and backend APIs.
Key Features
Converts live lecture audio into text notes automatically
Stores lecture notes in a structured database for later access
Backend APIs to process and manage lecture data
Enables easy retrieval of lecture notes for students
Technology Enhancement
Designed API based architecture for scalable lecture processing
Integrated database storage for persistent note management
Planned IoT hardware integration for automated classroom setup
Technologies Used
Backend: FastAPI
Database: MySQL
ORM: SQLAlchemy
Frontend (in progress): React.js
Hardware (planned): ESP32, Microphone, SD Card
Project Status
Backend API developed using FastAPI
Database connection implemented with MySQL and SQLAlchemy
Frontend development planned using React.js
Hardware integration planned with ESP32 and a microphone module
